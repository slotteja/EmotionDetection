# EmotionDetection
Completed May 2025, uploaded to GitHub Feb 2026

School final project to detect facial emotion using CNNs on FER2013 dataset (https://www.kaggle.com/datasets/msambare/fer2013)
Ran on Google Colab

'FER-CNN.ipynb' is main model + train/testing

'FER_CNN_GradCam/ValLoss.ipynb' is model + visualizations of facial features attention and outputs for presentation
